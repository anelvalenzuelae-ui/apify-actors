## 🛠️ Instalación

### Requisitos
- Node.js 14+
- Python 3.8+
- Apify CLI

### Pasos

1. Clonar el repositorio:
```bash
git clone https://github.com/anelvalenzuelae-ui/apify-actors.git
cd apify-actors
```

2. Instalar dependencias:
```bash
npm install -g apify-cli
pip install -r requirements.txt
```

3. Autenticarse en Apify:
```bash
apify login
```

4. Publicar un Actor:
```bash
cd linkedin-scraper
apify push
```

## 📈 Monetización

Todos los Actors están configurados para generar ingresos automáticos en Apify Store:

1. Los usuarios pagan por cada uso
2. Apify retiene el 20%
3. Tú recibes el 80% en tu cuenta PayPal
4. Pagos mensuales automáticos

## 🔐 Configuración de Pagos

1. Accede a [Apify Console](https://console.apify.com)
2. Ve a Settings → Billing
3. Configura tu método de pago (PayPal)
4. Los pagos se procesan entre el 15-22 de cada mes
